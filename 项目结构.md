## 🚀 快速开始

### 查看文档
```bash
# 项目概览
cat README.md

# 环境搭建
cat 开发环境搭建指南.md

# 部署指南
cat 部署指南.md

# 查看所有文档
ls docs/
cat docs/README.md
```

### 启动开发
```bash
# 1. 初始化数据库
mysql -u root -p < database/init.sql

# 2. 启动后端
cd mengyo-backend
mvn spring-boot:run

# 3. 启动管理后台
cd mengyo-admin
npm install && npm run dev

# 4. 启动小程序
cd mengyo-uniapp
npm install && npm run dev:mp-weixin
```

### 访问应用
- 后端API文档: http://localhost:8080/api/doc.html
- 管理后台: http://localhost:3000
- 小程序: 使用微信开发者工具打开 mengyo-uniapp
