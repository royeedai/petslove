- **边框色**: `#E8EAED`

## 🎯 设计原则

### 1. 色彩应用
- 主题色橘色贯穿全局
- 使用渐变色增强视觉层次
- 不同功能使用不同色彩区分

### 2. 间距和圆角
- 统一使用 24rpx/32rpx 圆角
- 卡片间距 20rpx/24rpx
- 内边距 28rpx/32rpx

### 3. 阴影效果
- 卡片阴影: `0 4rpx 20rpx rgba(0, 0, 0, 0.04)`
- 按钮阴影: `0 8rpx 24rpx rgba(255, 140, 66, 0.3)`
- 图标阴影: `0 4rpx 16rpx rgba(0, 0, 0, 0.08)`

### 4. 字体规范
- 大标题: 36rpx/32rpx (font-weight: 600)
- 正文: 28rpx/26rpx
- 辅助文字: 24rpx/22rpx

### 5. 交互反馈
- 按钮点击: `transform: scale(0.98)`
- 悬停效果: 背景色变化
- 动画时长: `0.3s ease`

```scss

  --primary-color: #FF8C42;
  --primary-gradient-start: #FF9D5C;
  --primary-gradient-end: #FF7F29;
  // ...更多变量
}
```

### 2. 渐变应用
```scss
background: linear-gradient(135deg, #FF9D5C 0%, #FF7F29 100%);
```

### 3. 阴影效果
```scss
box-shadow: 0 8rpx 32rpx rgba(255, 140, 66, 0.25);
```

### 4. 动画实现
```scss
@keyframes bounce {
  0%, 100% { transform: translateY(0); }
  50% { transform: translateY(-10rpx); }
}
```
