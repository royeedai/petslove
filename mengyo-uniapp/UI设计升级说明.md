# 萌友益站 - UI 设计升级说明

## 项目概述

本次对萌友益站 uniapp 移动端进行了全面的 UI 重新设计，采用**温暖治愈风格 (Style B: Warm & Cozy)**，提升用户体验和视觉美感。

---

## 设计理念

### 核心原则
- **温暖治愈**: 柔和的配色和大圆角设计，营造温馨友好的氛围
- **移动优先**: 针对移动端优化的布局和交互
- **视觉层次**: 清晰的信息架构和视觉引导
- **微交互**: 丰富的动画效果提升用户体验

---

## 设计规范

### 1. 配色方案

#### 主色调 - 温暖橙色系
- 主色: `#FF8C42` (暖橙)
- 渐变起始: `#FF9D5C`
- 渐变结束: `#FF7F29`
- 浅色变体: `#FFB88C`, `#FFDCC5`

#### 文字颜色 - 深褐色系
- 主文字: `#57534E` (stone-800 深褐)
- 次要文字: `#78716C` (stone-500)
- 辅助文字: `#A8A29E` (stone-400)
- 提示文字: `#D6D3D1` (stone-300)

#### 背景色 - 暖白米色系
- 页面背景: `#FFFBF5` (暖白/米色)
- 纯白: `#FFFFFF`
- 次要背景: `#FEF7F0` (浅米色)

#### 辅助色
- 成功: `#22C55E` (绿色)
- 警告: `#F59E0B` (琥珀色)
- 错误: `#EF4444` (红色)
- 信息: `#3B82F6` (蓝色)

### 2. 圆角系统 (超大圆角设计)
- 小圆角: `16rpx`
- 中圆角: `24rpx`
- 大圆角: `32rpx`
- 超大圆角: `40rpx`
- 全圆: `9999rpx`

### 3. 阴影系统
- 小阴影: `0 2rpx 8rpx rgba(255, 140, 66, 0.06)`
- 中阴影: `0 4rpx 16rpx rgba(255, 140, 66, 0.08)`
- 大阴影: `0 8rpx 32rpx rgba(255, 140, 66, 0.12)`
- 超大阴影: `0 16rpx 48rpx rgba(255, 140, 66, 0.15)`

### 4. 动画效果
- 淡入淡出 (fadeIn)
- 弹跳效果 (bounce)
- 脉冲动画 (pulse)
- 缩放效果 (scale)
- 旋转加载 (spin)

## 使用说明

### 1. 开发环境
```bash
# 安装依赖
npm install

# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin
```

### 2. 构建生产
```bash
# H5 构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin
```

### 3. 自定义样式
所有设计变量定义在 `App.vue` 的 `:root` 中，可以方便地修改全局配色：

```css
:root {
  --primary-color: #FF8C42;        /* 主色调 */
  --text-primary: #57534E;         /* 主文字颜色 */
  --bg-page: #FFFBF5;             /* 页面背景 */
  --radius-lg: 32rpx;             /* 大圆角 */
  --shadow-md: 0 4rpx 16rpx ...;  /* 中阴影 */
}
```
